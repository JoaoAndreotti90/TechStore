# 🛒 TechStore

Sistema Full Stack de e-commerce (loja virtual) desenvolvido com tecnologias modernas. O foco do projeto foi criar uma experiência de compra real, integrando pagamentos seguros, gerenciamento de produtos e autenticação.

## ✨ Funcionalidades

- 🔐 **Autenticação Segura:** Login rápido utilizando a conta do Google (OAuth 2.0).
- 💳 **Pagamentos Reais:** Integração com a API do Stripe para um checkout seguro e profissional.
- 🛒 **Carrinho de Compras:** Sistema completo para adicionar produtos, alterar quantidades e calcular o total.
- 👤 **Gestão de Perfil:** Área do usuário para atualizar dados de entrega e acompanhar pedidos.

## 🛠️ Tecnologias Utilizadas

**Front-end:**
- Next.js (React)
- Tailwind CSS
- TypeScript
- Zustand (Gerenciamento de estado global)

**Back-end & Banco de Dados:**
- NestJS
- Prisma ORM
- PostgreSQL 

**Integrações & Deploy:**
- Stripe API (Pagamentos)
- Google Cloud Console (Autenticação)
- Vercel (Deploy Front-end e Back-end)
